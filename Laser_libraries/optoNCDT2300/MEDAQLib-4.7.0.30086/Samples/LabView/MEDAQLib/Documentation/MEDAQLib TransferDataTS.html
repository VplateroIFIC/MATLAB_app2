Direction: [IN]

Description: Length of rawData and scaledData. It shoud be a multiple

of frame size, otherwise the rest of the last frame is lost, because

TransferData always starts with frame start.

instanceHandle

Parameter: DWORD instanceHandle

Direction: [OUT]

Description: Number of the sensor instance, copied from input.

DLL-Path

OUTPUT: Path to library "MEDAQLib.dll".

read

Parameter: int * read

Direction: [OUT]

Description: Will receive the real number of data values transfered. This

value returned will be less than the expected maxValues if not enough

data is available.

timestamp

Parameter: double * timestamp

Direction: [OUT]

Description: Will receive the timestamp of the first (oldest) value in data

array. It is in milli seconds starting at 01.01.1970 01:00.

It can be used to synchronize data from different sensors.

Each time when the internal ring buffer is cleared (e.g. at sensor

command), it is resetted and when next data block arrives it is

calculated by actual time minus aquire duration of data block (at

sensor). At TransferData it is automatically increased (by adding

duration of block, calculated from expected datarate). But if data

is discarded (because of the ring buffer overflow or synchronisation

lost), it is not incremented an therefore no longer correct. In this case,

call sensor command Clear_Buffers (8.2.7).

rawData

Parameter: int * rawData

Direction: [OUT]

Description: Pointer to array of values to retrieve data from sensor as raw values

    Parameter: int * rawData

    Direction: [OUT]

    Description: Pointer to array of values to retrieve data from sensor as

scaledData

Parameter: double * scaledData

Direction: [OUT]

Description: Pointer to array of values to retrieve data frame from sensor scaled by sensor range. If rawData is null or scaledData is null, no data is transferred for this parameter. If both parameters are null all buffers used are emptied.

    Parameter: double * scaledData

    Direction: [OUT]

    Description: Pointer to array of values to retrieve data frame from sensor

    scaled by sensor range.

    If rawData is null or scaledData is null, no data is transferred for this

    parameter. If both parameters are null all buffers used are emptied.

    The meaning of the raw and scaled values is described at each

    sensor section.
