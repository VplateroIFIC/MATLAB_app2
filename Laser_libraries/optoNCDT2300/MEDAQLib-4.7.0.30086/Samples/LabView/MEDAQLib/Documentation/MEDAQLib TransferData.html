MEDAQLib.lvlib:TransferData.vi

ERR_CODE TransferData (DWORD instanceHandle, int *rawData, double *scaledData, int maxValues, int *read);

Transfer the data from driver to application.

 instanceHandle

Parameter: DWORD instanceHandle

Direction: [IN]

Description: Number of the sensor instance, previously returned by CreateSensorInstance.

maxValues

Parameter: int maxValues

Direction: [IN]

Description: Length of rawData and scaledData. It shoud be a multiple

of frame size, otherwise the rest of the last frame is lost, because

TransferData always starts with frame start.

Fehler (Eingang, kein Fehler)

Fehler (Eingang) kann Fehlerangaben der vorherigen VIs aufnehmen. Auf Grundlage dieser Angaben kann entschieden werden, ob bei Entstehen eines Fehlers in einem anderen VI eine bestimmte Funktion übersprungen werden soll.

Für eine ausführlichere Erläuterung des Fehlers klicken Sie mit der rechten Maustaste auf das Anzeigeelement Fehler (Eingang) und wählen Sie aus dem Kontextmenü Fehler beschreiben oder Warnung beschreiben aus.

    		status

    Status ist TRUE (rotes Kreuz), wenn ein Fehler auftritt, andernfalls FALSE (grüner Haken).

    Für eine ausführlichere Erläuterung des Fehlers klicken Sie mit der rechten Maustaste auf das Element Fehler (Eingang) und wählen Sie aus dem Kontextmenü Fehler beschreiben oder Warnung beschreiben aus.

    		code

    Code ist der Fehler- oder Warnungscode.

    Für eine ausführlichere Erläuterung des Fehlers klicken Sie mit der rechten Maustaste auf das Element Fehler (Eingang) und wählen Sie aus dem Kontextmenü die Option Fehler beschreiben oder Warnung beschreiben aus.

    		source

    Quelle beschreibt den Ursprung des Fehlers oder der Warnung.

    Für eine ausführlichere Erläuterung des Fehlers klicken Sie mit der rechten Maustaste auf das Element Fehler (Eingang) und wählen Sie aus dem Kontextmenü Fehler beschreiben oder Warnung beschreiben aus.

DLL-Path

INPUT: Path to library "MEDAQLib.dll".

ERR_CODE

Returns:

ERR_NOERROR (0) on success.

ERR_INSTANCE_NOT_EXTST (-24) if instanceHandle is not valid.

ERR_NOT_OPEN (-3) if connection is not established

ERR_NO_SENSORDATA_AVAILABLE (-13) if the receive buffer does not contain any value from sensor.

rawData

Parameter: int * rawData

Direction: [OUT]

Description: Pointer to array of values to retrieve data from sensor as raw values

scaledData

Parameter: double * scaledData

Direction: [OUT]

Description: Pointer to array of values to retrieve data frame from sensor scaled by sensor range. If rawData is null or scaledData is null, no data is transferred for this parameter. If both parameters are null all buffers used are emptied.

read

Parameter: int * read

Direction: [OUT]

Description: Will receive the real number of data values transfered. This

value returned will be less than the expected maxValues if not enough

data is available.
