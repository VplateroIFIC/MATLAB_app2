MEDAQLib.lvlib:CreateSensorInstance.vi

DWORD CreateSensorInstance (ME_SENSOR sensor);

Creates an instance for the sensor which will be used in further function calls.

Attention! Please don't forget to release the instance at the end. Otherwise

there will be memory and handle leaks.

Parameter: ME_SENSOR sensor
