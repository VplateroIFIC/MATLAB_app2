MEDAQlib Instrument Driver Readme

1. Overview

Instrument Driver Technology: LabVIEW Plug and Play (project-style)
Manufacturer: Micro-Epsilon Messtechnik GmbH&Co.KG
Supported Language(s): English
Supported Model(s):
  • ILR sensors: optoNCDT110x, optoNCDT115x, optoNCDT118x, optoNCDT1191.
  • ILD sensors: optoNCDT1302, optoNCDT1401, optoNCDT1401-Controller, opto-NCDT1402, optoNCDT1700, optoNCDT1800, optoNCDT2000, optoNCDT2200 (including optoNCDT2220), optoNCDT2300.
  • IFD sensors: optoNCDT2400, optoNCDT2401, optoNCDT2430, optoNCDT2431, confocalDT2451, confocalDT2471.
  • ODC sensors: optoCONTROL1202, optoCONTROL2500, optoCONTROL2520, optoCONTROL2600.
  • Eddy sensors: eddyNCDT3100, eddyNCDT3909, eddyNCDT4912.
  • Capa sensors: capaNCDT6100, capaNCDT6200, capaNCDT6500.
  • Interface cards: IF2004, IF2008, IF2004_USB, C-Box, CSP2008.

2. Installation

  • Copy the folder <MEDAQLib install path>\Samples\LabView\MEDAQLib into <LabVIEW install path>\instr.lib (e.g. C:\Program Files\National Instruments\LabVIEW 8.6\instr.lib).
  • Ensure that MEDAQLib.dll exists in the subfolder <LabVIEW install path>\instr.lib\MEDAQLib\Release.
  • Perform a mass compile in LabVIEW to re-link the VIs an avoid inconsistencies (Tools --> Advanced -> Mass Compile.. and select the extracted folder MEDAQLib in the instr.lib folder).
  • The driver package appears in your Functions palette.
